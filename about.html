About the site

A Learning Project

This color gradient blog is a learning project as I aspire to become a front end web developer. I tried to exercise a good set of skills and practices in putting this together.

Hello, my name is Sin Ho Ching, Victoria. I come from Hong Kong, and I just immigrated into U.S.A.

Over my job search, I decided to learn a new skill - Programming!

This project was prompted by the Udacity Nanodegree course I'm currently taking. This site uses CSS grid with media queries to be responsive.

And do not forget the beauty of brilliant gradients! Learning aside, the true purpose of this site is to bring gradients to the world. I think they make wonderful filler content for a learning project site such as this. Bon appetit!
